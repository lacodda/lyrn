## Description
Lyrn is command line tool for rapid starting the development of your web application

## Installation

You can install the package from npm

```bash
npm i -g lyrn
```

## Usage

```bash
lyrn download https://github.com/lacodda/lyrn.git project-name
```

## Tests

```
npm test
```
